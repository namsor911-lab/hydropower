## ✨ WHAT HAPPENS ON DEPLOYMENT

```
You run: deploy-railway.sh (or .bat)
    ↓
Railway CLI authenticates
    ↓
Uploads code to Railway
    ↓
Railway detects Dockerfile
    ↓
Railway builds Docker image (Alpine Node 18)
    ↓
Railway starts container
    ↓
Container runs: npm run start
    ↓
backend/src/setup-production.js executes:
    ✅ Initializes SQLite database
    ✅ Creates 8 tables (users, transactions, etc.)
    ✅ Creates admin user (admin@namsor.local)
    ✅ Creates accountant user
    ✅ Creates viewer user
    ✅ Shows credentials
    ↓
Express server starts on port 5000
    ↓
Frontend files served at root
    ↓
Your app is LIVE! 🎉
```
